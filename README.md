# Task-Manager-App

## Description
A simple full stack task manager application that allows users to create, view, update, and delete tasks.

##Tech Stack
- Backend: Spring Boot
- Frontend: HTML, JavaScript

##Features
- Add new task
- View all tasks
- Mark task as complete
- Delete task

## How to Run

### Backend
1. Open project in IntelliJ
2. Run TaskManagerApplication

### Frontend
1. Open index.html in browser

## Notes
- Used in-memory storage (no database)
- Focused on clean structure and working API
